# 记账本 💰

一个简洁高效的个人记账网站，移动端优先设计，支持离线使用。

## 功能特点

- 📝 **快速记账** - 数字键盘快速输入，一键保存
- 📂 **分类管理** - 15+ 预设分类，支持自定义
- 💳 **多账户** - 现金、支付宝、微信、银行卡等
- 📊 **统计报表** - 饼图、趋势图直观展示收支
- 💵 **预算管理** - 设置月度预算，超支提醒
- 📤 **数据备份** - JSON 导入导出，数据永不丢失

## 在线访问

👉 [点击访问](https://zhaoluhang1989.github.io/expense-tracker/)

## 技术栈

- HTML5 + CSS3 + JavaScript (原生)
- Chart.js 图表库
- LocalStorage 本地存储
- Mobile-first 响应式设计

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
